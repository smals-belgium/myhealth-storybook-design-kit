# Typography

> Typography can help convey hierarchies, organising information and guiding users throughout a product or experience.

## Typography tokens

My Health uses typography tokens to manage typography. Typography tokens are style configurations of typographic elements such as font size, weight or leading (line-height). These pre-set styles are linked with specific uses in the interface.

## Typeface: Open Sans

My Health uses the open-source typeface Open Sans which is provided by [Google fonts](https://fonts.google.com).

### Use on the web

The **Opens Sans**, **Open Sans Italic** and **Material Symbols Rounded** fonts have been embedded into the design kit and are installed automatically.

A full guide on how to use Material Symbols with Angular Material can be found in their [component documentation](https://material.angular.io/components/icon).

### Open Sans family

Open Sans is My Health's primary typeface. It's used for plain text `<body>`. The Design Kit uses predefined mixins you can use for all headings, body text, links and labels.

### Usage

To use these mixins, you can simply include them into your SCSS file. Here is an example:

```css
@use '@myhealth-belgium/design-kit' as mh;

p {
  @include mh.font-body-4();
}
```

## Style

Typography creates a structure that guides users to reading and understanding the information hierarchy. The right typographic scale and use of type styles helps create clear distinctions, keeping content readable, simple and effective.

### Weights

Font weight is an important typographic variable that can add emphasis and distinguish content hierarchy. Font weight and size of the typeface must be well-balanced. A bold weight will always have more emphasis than a lighter weight font of the equal size. However, a lighter weight font may rank higher than a bold font if the smaller weight type is much larger than the bold font.
